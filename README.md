# Student Management System

A simple **Student Management System** developed in **C++** that stores student records using **file handling**. The project demonstrates the use of object-oriented programming concepts and persistent data storage without requiring a database.

##  Features

-  Add new student records
-  Search student details
-  Update existing records
-  Delete student records
-  Display all student records
-  Persistent storage using file handling

##  Technologies Used

- C++
- File Handling (fstream)
- Object-Oriented Programming
- STL 

## Project Structure

```
student-management-system/

```


##  How to Run

1. Clone the repository

```bash
git clone https://github.com/harshita07agr-source/student-management-system.git
```

2. Open the project in your preferred C++ IDE (VS Code, CodeBlocks, Dev-C++, etc.).

3. Compile the program.

Example using g++:

```bash
g++ student.cpp -o student
```

4. Run the executable.

```bash
./student
```
